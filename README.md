# developKit

### 使用方法
##### 1、Add it in your root build.gradle at the end of repositories:
##### allprojects {
##### 		repositories {
##### 			...
### 			maven { url 'https://jitpack.io' }
##### 		}
##### 	}
  
##### 2、Add the dependency
##### dependencies {
### 	        implementation 'com.github.fe203.developkit:business:0.12'
##### 	}
